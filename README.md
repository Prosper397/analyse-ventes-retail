# Analyse des ventes - Online Retail

## Objectif
Identifier les produits, périodes et marchés les plus rentables
à partir de 270 000 transactions e-commerce réelles (2010-2011).

## Principaux insights
- Le REGENCY CAKESTAND 3 TIER est le produit star (£115 728)
- Novembre 2011 = mois record avec £952 838 de revenu
- Le Royaume-Uni représente ~85% du revenu total
- Tendance haussière claire de juillet à novembre (saisonnalité)

## Outils
Python · Pandas · Matplotlib · Seaborn · Google Colab

## Étapes
1. Chargement et exploration des données
2. Nettoyage (valeurs nulles, retours, prix aberrants)
3. Analyse (groupby par produit, mois, pays)
4. Visualisation (bar chart, line chart)
